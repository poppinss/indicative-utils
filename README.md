## Usage
Install the package from npm registry as follows:

```sh
npm i indicative-utils

# yarn
yarn add indicative-utils
```

and then use it as follows:

```ts
import {
  cast,
  ensureLength,
  getValue,
  patchValue,
  skippable
} from 'indicative-utils'
```
